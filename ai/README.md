## Agent generator

Example on https://chatgpt.com/share/690dfd40-02a4-8006-9d24-468103339f0b

Optimize an AI Agent instructions for the persona <PERSONA>.

- Each Instructions block should be optimized for both GPT-5 model, claude Sonnet 4.5 and gemini Pro 2.5. 
- Keep it under 700 tokens (or 2500 characters).
- Don't mention lando.
- Focus on Ubuntu development.
- Focus on LAMP stack.
- Title should be h1, sections should be h2.
- Title should be short, ready to be used on a prompt UI.
- No lines between sections.
